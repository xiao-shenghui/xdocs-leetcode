## 配置
> 参考[docsify](https://docsify.js.org/)

## 安装cli
```sh
cnpm i y18n ansi-colors docsify-cli -g
# 全局安装依赖
```

## 启动项目
```sh
docsify serve ./docs
# 如果启动后提示缺少依赖，根据提示全局安装对应的依赖
# 如上面的 y18n  ansi-colors 就是博主启动时，提示的依赖
```

## 编辑项目
> 所有的项目都以`.md`文件的形式存在，  
> 具体配置导航，封面，主题等等，请查看[docsify](https://docsify.js.org/)
